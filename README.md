# verified-instances

Verified typeclasses using [LiquidHaskell](http://goto.ucsd.edu/~rjhala/liquid/haskell/blog/about/)

### Run with LH-0.8.4

```
$ git clone --branch lh-84 git@github.com:ranjitjhala/verified-instances.git 
$ cd verified-instances 
$ stack build 
$ ./liquid.sh
```
